# BizManage - Project Description

## Short Description (for GitHub repo)

**BizManage ERP** — A comprehensive Enterprise Resource Planning system for trading companies with invoicing, payments, VAT compliance, multi-company support, role-based access control, and modern React + Express.js stack.

## One-Liner

Modern ERP platform for Sri Lankan businesses: manage invoices, payments, inventory, and multi-company operations with RBAC and JWT authentication.
